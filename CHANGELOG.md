# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.1] - 2021-03-27
### Fixed
- Fixed the `Comparison 'success == False' should be 'success is False' if checking for the singleton value False, or 'not success' if testing for falsiness` issue.

## [1.2.0] - 2021-03-24
### Removed
- We removed `i`, `w`, `e`, and `inf`

## [1.1.0] - 2021-03-23
### Changed
- Now you don't have 3 parameters in the `log()` function, now you have one! (`log({"type": "info", "text": "RickRoll", "return": "TRUEFALSE"})`)
